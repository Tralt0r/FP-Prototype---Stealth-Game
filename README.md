# FP-Prototype---Stealth-Game
Repository focusing on our stealth game for project 2.

Controls:

For Keyboard and Mouse

WASD - Movement

Spacebar - Jump

Left Shift - Sprint

E - Interact

Left Mouse Button - Click Menu

Move Mouse - Look Around

Esc - Pause



For Controllers

Left Stick - Movement

South Button - Jump

Left Stick Pressed - Sprint

West Button - Interact and Click Menu

Right Stick - Look Around

Start - Pause
